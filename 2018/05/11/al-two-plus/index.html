两数相加

2018-05-11
LeetCode

题目

给定两个非空链表来表示两个非负整数。位数按照逆序方式存储，它们的每个节点只存储单个数字。将两数相加返回一个新的链表。
你可以假设除了数字 0 之外，这两个数字都不会以零开头。

示例：
输入：(2 -> 4 -> 3) + (5 -> 6 -> 4)
输出：7 -> 0 -> 8
原因：342 + 465 = 807

我的解法 34ms 战胜96.11%的java 提交记录

核心思路
  • 两个数与进位相加
  • %10余数为当前节点值
  • /10除数为进位值
  • 循环条件：两个链表一个不为空，或carry大于0遍
